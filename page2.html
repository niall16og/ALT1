Research and Investigation

Embedded systems are designed to carry out specific tasks as either an independent system or part of a larger system. They are used in many instances in both school and home environments. Some examples are:


Description Image

Household Appliances - Many household appliances contain embedded systems, such as washing machines and dishwashers.

Alarm Security Systems - Alarm security systems use embedded systems to carry out functions, such as arming and disarming the alarm and setting the alarm based on inputs received from sensors or buttons/switches on the alarm console.

Classroom CO2 Monitors - Many schools have CO2 monitors in their classrooms. These run on embedded systems that can output the CO2 levels with a colour code on a display based on information read by sensors.

Through these examples of embedded systems that we found, we were able to get an idea of what kind of project we wanted to make and how we would have to deal with inputs correctly in order to produce the desirable functional output.
